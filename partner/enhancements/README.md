# Partner Enhancements

Optional extensions to `duran-serkan.md` based on research of production AI agent prompts.

## Files

| File | Purpose |
|------|---------|
| `session-protocol.md` | First-turn handling and session priming |
| `state-preservation.md` | Multi-turn context and memory management |
| `panel-synthesis.md` | Multi-perspective reasoning for critical decisions |
| `worked-examples.md` | Concrete scenarios showing persona in action |
| `advanced-reasoning.md` | Tree-of-Thoughts, Reflexion, CodeAct patterns |

## Usage

These can be:
1. **Merged** into `duran-serkan.md` as new sections
2. **Referenced** as supplementary guidance when needed
3. **Loaded conditionally** based on task complexity (dynamic assembly pattern)
